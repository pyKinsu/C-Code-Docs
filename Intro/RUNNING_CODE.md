# How to Run C Programs

This guide explains how to run C programs on different systems.

---

## Windows (Code::Blocks)

1. Open **Code::Blocks**
2. Click **File → New → Empty File**
3. Write or paste your C code
4. Save the file with `.c` extension (e.g., `program.c`)
5. Click **Build and Run** (or press `F9`)
6. Output will be displayed in the console window

---

##  Linux

```bash
gcc program.c -o program
./program
````

---

##  macOS

```bash
gcc program.c -o program
./program
```

---

## 📱 Android (Optional)

* Use apps like **Cxxdroid** from Play Store
* Write code and run directly on your phone
