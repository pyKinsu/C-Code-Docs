# Your First C Program

Write and run your first C program — the classic **Hello, World!**

This is just for your curiosity. In the next sections, we will learn concepts like `#include`, `main()`, `printf()`, and basic syntax in detail.

---

## 📌 Program

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
````

---

## ▶️ How to Run (Code::Blocks - Windows)

1. Open **Code::Blocks**
2. Click on **File → New → Empty File**
3. Paste the above code
4. Save the file as `program.c`
5. Click **Build and Run** (or press `F9`)

---

## 🖥 Output

```
Hello, World!
```

---

## 🧠 Explanation

* `#include <stdio.h>` — Includes input/output functions
* `int main()` — Starting point of the program
* `printf()` — Displays output
* `return 0;` — Ends the program

---

## ⚠️ Common Mistakes

* Missing `;` at the end of statements
* Forgetting `#include <stdio.h>`
* Writing `print` instead of `printf`
